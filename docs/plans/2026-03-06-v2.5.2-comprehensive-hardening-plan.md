# v2.5.2 Comprehensive Robustness Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use `superpowers:executing-plans` to implement this plan task-by-task.
> **For Cursor:** Review each task for missing edge cases and test gaps before coding starts.

**Goal:** Ship a single consolidated v2.5.2 release that comprehensively hardens troubleshooting, diagnostics, edge-case handling, and recovery paths for the PSEG integration + add-on.

**Architecture:** Keep the existing split (HA integration + Playwright add-on), but make failure handling deterministic end-to-end: add-on emits structured failure signals and persistent artifacts, integration consumes those signals, classifies remediations, and exposes one diagnostic payload for operators. Emphasize bounded retries, stable state transitions, and debuggability with low operational noise in steady state.

**Tech Stack:** Home Assistant custom component (Python, asyncio, aiohttp), Home Assistant add-on (FastAPI, Playwright), pytest/pytest-asyncio.

## Scope

In scope:
- Unified failure taxonomy and structured diagnostics across add-on + integration
- Persistent artifact capture and retrieval
- Robust CAPTCHA vs invalid credentials vs site-flow-change classification
- Connectivity/URL diagnostics and recovery clarity
- Scheduler/backfill/retry hardening under edge conditions
- Debug mode lifecycle controls (enable for troubleshooting, auto-disable after stabilization)
- Comprehensive deterministic test coverage and release docs

Out of scope:
- New external services
- Non-PSEG domain integrations
- Major UI redesign in Home Assistant frontend

## Release Criteria (must all pass)

- Stable suite passes (`pytest -q`) with no new warnings/errors
- HA-recent lane passes or only known, documented harness deltas
- Every failure category has deterministic tests and user-facing remediation text
- Add-on artifacts persist under `/data` and are retrievable after restart
- `psegli.get_status` + diagnostics payload include enough context for first-line triage
- CHANGELOG + release notes accurately summarize behavior changes
